# omarchy-timer
Adds a minimal timer TUI utility, which integrates with waybar and has the ability to do pomodoro-style timers.

> [!IMPORTANT]
> This utility is Omarchy specific and depends only on packages, which are already pre-installed on Omarchy, meaning no additional dependencies.

## Demo
https://github.com/user-attachments/assets/fbf8f58f-75bb-450d-9cd3-4e1d9c1f1062

## Installation
```bash
git clone git@github.com:severin-richner/omarchy-timer.git && chmod +x ./omarchy-timer/omarchy-timer && ./omarchy-timer/omarchy-timer install
```

## Uninstallation
Run `omarchy-timer uninstall` and follow the steps.
